Link one stylesheet, write tags

A render-blocking stylesheet styles the tags before first paint. Works in plain HTML or any framework — server-render the markup and it just works.

<link rel="stylesheet" href="https://unpkg.com/layoutkit-css/layoutkit.css">

<lk-stack gap="lg" padding="md">
  <lk-center full-height>You can finally center a div.</lk-center>
  <lk-grid cols="3" gap="md">...</lk-grid>
</lk-stack>
