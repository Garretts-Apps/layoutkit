Install LayoutKit

Link the stylesheet from a CDN, vendor it locally, or install the CSS package from npm.

Link the stylesheet

LayoutKit is just CSS. Add one render-blocking <link> and write semantic custom tags — no JavaScript, no build, no FOUC: 

<link rel="stylesheet" href="https://unpkg.com/layoutkit-css/layoutkit.css">

<lk-stack gap="lg" padding="md">
  <lk-row gap="sm" justify="between">
    <span>Logo</span>
    <nav>…</nav>
  </lk-row>
  <lk-center full-height>You can finally center a div.</lk-center>
  <lk-grid cols="3" gap="md">
    <div>1</div><div>2</div><div>3</div>
  </lk-grid>
</lk-stack>

Pure CSS, zero dependencies, no JavaScript runtime, no build step. ~2 KB brotli.

npm (CSS package)

# it's a CSS package — import the stylesheet
npm install layoutkit-css

It's a CSS package — import the stylesheet once, then write <lk-*> tags anywhere: 

import "layoutkit-css/layoutkit.css";

<lk-stack gap="lg" padding="md">
  <lk-row gap="sm" align="center">
    <h1>Hello</h1>
    <span>World</span>
  </lk-row>
  <lk-center fill>Centered content</lk-center>
  <lk-grid cols="3" gap="md">
    <div>1</div><div>2</div><div>3</div>
  </lk-grid>
</lk-stack>

A single CSS import — no JavaScript, no Tailwind, no build step, no config.

Vendor it — no package manager, no build, no JavaScript

One pure-CSS stylesheet. Drop a single <link> tag into any HTML page — or any framework — and write the <lk-*> tags. No npm, no bundler, no JavaScript runtime. You own the file and can edit it freely.

Use it straight from the CDN — nothing to install:

<link rel="stylesheet" href="https://unpkg.com/layoutkit-css/layoutkit.css">

<lk-stack gap="lg" padding="md">
  <lk-row gap="sm" justify="between">
    <span>Logo</span>

  </lk-row>
  <lk-center full-height>You can finally center a div.</lk-center>
  <lk-grid cols="3" gap="md">
    <div>1</div><div>2</div><div>3</div>
  </lk-grid>
</lk-stack>

Ten semantic custom tags (lk-stack, lk-row, lk-center, …) styled by attribute selectors. Attributes are kebab-case (full-height, col-gap); free-form values use inline custom properties (--lk-min-child-width). Pure CSS, zero dependencies, no FOUC, ~2 KB brotli.

Attributes & values

Attributes are kebab-case. The full set: gap, padding, align, justify, center, fill, full-height, wrap, reverse, cols, col-gap, row-gap, responsive, size, orientation, thickness, direction.

Free-form values

For values outside the preset scale, set an inline custom property:

--lk-ratioCustom aspect ratio (e.g. 16/9)
--lk-divider-colorDivider line color
--lk-min-child-widthMin child width for responsive grids
--lk-max-heightScroll-area max height
--lk-max-widthScroll-area max width
